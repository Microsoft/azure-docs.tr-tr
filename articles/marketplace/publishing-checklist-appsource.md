# <a name="publishing-checklist-for-appsource"></a>AppSource için yayımlama denetim listesi

Uygulamanızı AppSource 'ta yayımlamak için aşağıdaki yapıtlar gereklidir.  

## <a name="checklist"></a>Denetim Listesi
