# <a name="additional-documentation"></a>Ek belgeler 

Bu docs.microsoft.com Web sitesinin dışındaki bazı kaynaklar Microsoft Azure FXT Edge Filer karma önbelleğinizi anlamanıza ve bunlarla çalışmanıza yardımcı olur.

## <a name="avere-vfxt-for-azure-cache-documentation"></a>Azure önbellek belgeleri için avere vFXT

[Azure Için avere vFXT](https://docs.microsoft.com/azure/avere-vfxt/) , Microsoft Azure içindeki sanal düğümleri kullanan benzer bir küme önbelleği ürünüdür. Azure portal ile oluşturulur, ancak Azure FXT Edge Filer donanım önbelleği ile aynı denetim masası yapılandırma yazılımını kullanır. 

## <a name="legacy-hardware-documentation"></a>Eski donanım belgeleri 

Bu teknoloji daha önce avere Systems ürünlerinde kullanılmıştı. Eski ürünlere yönelik kullanıcı belgeleri [avere eski belgeler Web sitesinde](https://azure.github.io/Avere/)bulunabilir. 

Özellikle, bu belgelerde yararlı ayrıntılar olabilir:

* [Küme yapılandırma kılavuzu](https://azure.github.io/Avere/legacy/ops_guide/4_7/html/ops_conf_index.html) -ortak küme kurulumu ve yönetim görevleriyle ilgili genel bilgiler içerir ve küme ayarları sayfalarına ilişkin ayrıntılı açıklamalar içerir 

* [Pano Kılavuzu](https://azure.github.io/Avere/legacy/dashboard/4_7/html/ops_dashboard_index.html) -Denetim Masası **Pano** sekmesinin özelliklerini açıklar

* [FXT kümesi oluşturma kılavuzu](https://azure.github.io/Avere/legacy/create_cluster/4_8/html/create_index.html) -önceki ürünlerden küme oluşturma kılavuzu
