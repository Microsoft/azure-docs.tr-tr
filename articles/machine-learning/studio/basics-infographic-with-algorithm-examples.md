# <a name="downloadable-infographic-machine-learning-basics-with-algorithm-examples"></a>İndirilebilir Bilgi Grafiği: Algoritma örnekleri ile machine learning temel bilgileri

Ortak makine öğrenimi sorularını yanıtlamak için kullanılan popüler algoritmaların öğrenilmesi için, Machine Learning temel bilgileri ile ilgili kolay anlaşılır bilgi grafiğine genel bakış konusunu indirin. Algoritma örnekleri, Machine Learning 'in hangi algoritmaların kullanılacağını ve için ne kullanıldığını anlamasına yardımcı olur.

## <a name="popular-algorithms-in-machine-learning-studio-classic"></a>Machine Learning Studio popüler algoritmalar (klasik)

[Azure Machine Learning Studio (klasik)](https://studio.azureml.net/) , tahmine dayalı analiz için büyük bir algoritma kitaplığıyla birlikte gelir. Bu Infographic, dört popüler algoritma ailesi (gerileme, anomali algılama, kümeleme ve sınıflandırma) tanımlar ve [Azure yapay zeka Galerisi](https://gallery.azure.ai/)çalışan örneklere bağlantılar sağlar. Galeri, bu algoritmaların birçok gerçek dünya çözümlerinde nasıl uygulanabileceğini gösteren örnek denemeleri ve öğreticiler içerir.

## <a name="more-help-with-algorithms-for-beginners-and-advanced-users"></a>Yeni başlayanlar ve gelişmiş kullanıcılara yönelik algoritmalarla ilgili daha fazla yardım

* Farklı makine öğrenimi algoritmalarının, nasıl kullanılabilecekleri ve çözümünüz için doğru bir şekilde nasıl seçeceğiniz hakkında daha derin bir tartışma için bkz. [Microsoft Azure Machine Learning Studio algoritmaları seçme (klasik)](algorithm-choice.md).
* Machine Learning Studio klasik sürümünde bulunan tüm makine öğrenimi algoritmalarının kategorilerine göre bir liste için, bkz. Machine Learning Studio (klasik) algoritması ve modül yardımı 'nda [modeli başlatma][initialize-model] .
* Machine Learning Studio klasik sürümündeki algoritmaların ve modüllerin tam alfabetik listesi için, Machine Learning Studio (klasik) algoritması ve modül yardımı 'nda [a-Z Machine Learning Studio (klasik) modülleriyle Ilgili bir][a-z-list] liste bölümüne bakın.
* Azure Yapay Zeka Galerisi ve topluluk tarafından üretilen birçok kaynağa genel bir bakış için, bkz. [Azure yapay zeka Galerisi kaynakları paylaşma ve bulma](gallery-how-to-use-contribute-publish.md).
