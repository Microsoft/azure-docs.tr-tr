Komut çıkışında, `identity` bölümünde türünde `SystemAssigned` bir kimlik ayarlanır. , `principalId` Kimliğin hizmet sorumlusu kimliğidir:

```console
[...]
  "identity": {
    "principalId": "xxxxxxxx-2703-42f9-97d0-xxxxxxxxxxxx",
    "tenantId": "xxxxxxxx-86f1-41af-91ab-xxxxxxxxxxxx",
    "type": "SystemAssigned",
    "userAssignedIdentities": null
  },
  "location": "eastus",
[...]
``` 
Daha sonraki komutlarda kullanmak için PrincipalId 'yi bir değişkende depolamak üzere [az ACR Task Show][az-acr-task-show] komutunu kullanın:

```azurecli
principalID=$(az acr task show --name dockerhubtask --registry myregistry --query identity.principalId --output tsv)
```
