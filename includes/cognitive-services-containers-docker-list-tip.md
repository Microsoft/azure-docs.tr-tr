> [!TIP]
> İndirilen kapsayıcı görüntülerinizi listelemek için [Docker görüntüleri](https://docs.docker.com/engine/reference/commandline/images/) komutunu kullanabilirsiniz. Örneğin, aşağıdaki komut, indirilen her bir kapsayıcı görüntüsünün KIMLIĞINI, deposunu ve etiketini listeler ve tablo olarak biçimlendirilir:
>
>  ```
>  docker images --format "table {{.ID}}\t{{.Repository}}\t{{.Tag}}"
>
>  IMAGE ID         REPOSITORY                TAG
>  <image-id>       <repository-path/name>    <tag-name>
>  ```
